## Matplotlib

This folder documents my hands-on learning of **Matplotlib**, focusing on fundamental data visualization techniques in Python.  
All examples are created and tested in **Jupyter Notebook** as part of my internship learning.

---

### Covered Topics
- Line plots using `plt.plot()`
- Markers, line styles, and colors
- X-axis and Y-axis labeling
- Plot titles and basic customization
- Understanding plot output behavior in Jupyter Notebook
- Bar charts
- Histograms
- Scatter plots
- Multiple plots and subplots
- Legends, grids, and styling
